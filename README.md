# `ginger` monorepo

## Links

- `packages/ginger`: [README.md](./packages/ginger/README.md)

## Common Commands

- Verify everything locally (same order as CI): `npm run verify`
- Demo app: `npm run dev`
- Landing app: `npm run dev:landing`

## Apps

- `apps/ginger-landing`: Landing page with an editor-style tooltip on the `Ginger` title that previews a minimal Ginger player snippet on hover/focus.
